# 🚀 SwiftDrop

SwiftDrop is a blazing-fast, no-BS file sharing app for Android, built using **Expo + BLE + TCP**.  
It combines modern design with real peer-to-peer file transfers — no ads, no spyware, no fake "cleaners" or shady stuff. Just powerful, sleek file sharing.

---

## ✨ Features

- 📡 **Real Bluetooth (BLE) device discovery**  
- 🔄 **Peer-to-peer file transfer over TCP sockets** (local Wi-Fi or hotspot)
- 🔍 Radar-style UI for finding nearby devices
- 📷 **QR Code mode** for instant connections
- 🗂 Built-in **file manager**: Photos, Videos, Documents, Apps & APKs
- 🎯 Sleek dark UI with animated glows (no neon junk)
- ⚡ Fully offline — no internet required to send files
- 🔐 Secure — no cloud upload, nothing leaves your device

---

## 📱 Tech Stack

- **Expo + Expo Dev Client**
- **React Native (TypeScript)**
- [`react-native-ble-plx`](https://github.com/dotintent/react-native-ble-plx) for Bluetooth
- [`react-native-tcp-socket`](https://github.com/Rapsssito/react-native-tcp-socket) for file transfer
- `expo-media-library`, `expo-camera`, `react-native-qrcode-svg` for file access and QR
- `nativewind` for styling
- Cursor.dev + Bolt.new for AI-assisted development & design
